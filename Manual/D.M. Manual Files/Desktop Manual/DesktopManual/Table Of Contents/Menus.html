List of Desktop Menus

This chapter contains a list of the menus and menu items which are accessed from the standard Macintosh menu bar. The menu items are listed with the corresponding menu and contain a brief description of the action or function activated by their selection. If the menu item activates a dialog box, the dialog box is referenced with a brief description of its use and its location in the chapter on Dialog Boxes (Chapter VI). If the action activates a function, such as rotation of the stereogram, the description includes an outline of how to use the function.
