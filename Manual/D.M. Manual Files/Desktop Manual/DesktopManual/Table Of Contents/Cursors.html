Desktop Microscopist Cursors

Cursors & Miscellaneous;:
This chapter contains a list of the cursors and some of the important figures which appear on the Desktop Microscopist screen. It has been included to support a complete reference of all actions available to the Desktop Microscopist user.
  • The arrow cursor:
    I. Used to point and click on buttons, menus, diffraction spots, etc. The active spot is the tip of the arrow.

    Arrow Cursor
  • Gun Sight Cursor
    The gun sight cursor is activated when using a function which requires the selection of an object before the desired action can occur. The active region of the gun sight cursor is at its center.

  • Zoom In Cursor
    The zoom in cursor in the active function will magnify the current plot. There are two methods for achieving magnification.
    1. · Clicking on the screen with this cursor will cause the image to magnify by 2x around the click point.
    1. · Click, hold the mouse down and drag. The movement of the mouse causes a rectangle to appear. Releasing the mouse will cause the region inside the rectangle to fill the entire screen.


  • Bulls Eye Cursor
    The bull's eye cursor is seen in conjunction with a centering circle and either a straight or curved line. The bull's eye cursor is used in the experimental section in a fashion that is similar to the gun sight cursor. A unique feature of the bull's eye cursor is that the radius of the centering circle is variable. The Centering ring radius is dependent on the defined convergence angle of the current beam. Increasing the convergence angle will increase the radius of the centering ring. The bull's eye cursor is for extracting experimental information from a diffraction pattern. Usually, it is easier to extract this information when the radius of the ring corresponds to the intensity of the diffracting spots.

  • Watch Cursor
    The watch cursor is used to signify: "Please wait...Calculation in Progress". Usually, when this watch comes on Desktop Microscopist can be put into the background.

  • Ruler Buttons

    • Desktop Microscopist supports four different types of buttons for the ruler. Each button is selected using the arrow cursor and displays a pop-up menu of the relevant information.
    • The .i.Ruler Button:Text Object; button enables or disables the screen text entry option. This option allows the user to randomly place text and modify the size, style, color and font of that text. .c3.VI.B.1 Text;
    • The .i.Ruler Button:Crystal; buttons display information on the specific crystals. Double clicking on this ruler button will bring up the correct Crystal Definition Dialog Box..c3.VI.B.2 Crystals;
    • The .i.Ruler Button:X-ray; or .i.Ruler Button:Electron ;button displays information on the specified high energy beam. The information supplied is for the beam used to calculated the current diffraction plot. Double clicking on this ruler button will bring up the corresponding Beam Definition Dialog Box. .c3.VI.B.3 X-ray or Electron;
    • The .i.Ruler Button:Define ;button displays the SetUp or Definition information for the current plot. This information includes the beam orientation (if relevant), the color of each crystal plotted and other miscellaneous information. Double clicking on this ruler button will bring up the Definition Dialog Box for the object. .c3.VI.B.4 Define;
