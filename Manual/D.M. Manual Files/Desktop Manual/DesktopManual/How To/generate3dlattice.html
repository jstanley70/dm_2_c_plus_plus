Generating A 3-Dimensional Lattice.


Generating a three dimensional lattice using desktop microscopist is simple if the user has followed the directions for properly generating a set of 2 or more 2-d primitive lattices. Once two or more 2-d primitive lattices are generated from a given crystal and the tilts between the 2-d lattices are recorded, go to the Definition Menu in the 2-D Lattice Table Display Object. This will bring up the 2D->3D Lattice Generation Dialog Box. Simply selecting the two or more 2-d lattices and clicking on the Calculate button will generate a three dimensional lattice.
