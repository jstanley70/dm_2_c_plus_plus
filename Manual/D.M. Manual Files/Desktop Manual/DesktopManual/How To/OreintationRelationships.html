How to Determine An Orientation Relationship

There are two distinct methodologies for working out an orientation relationship between two crystals. The method is based on trial and error and requires the use of the Precipitate Info Dialog Box to set up a predicted orientation relationship and then the use of the appropriate diffraction simulation package to generate various diffraction patterns. The user then compares the simulated pattern to the experimental relationship.
The second method uses Desktop Microscopist's ability to extract orientation information from an experimental SAD diffraction pattern.
