How To Determine Experimental Tilts

One of the standard road blocks to accurate and fast cyrstallography is the difficulty in tilting to a new specified orientation. These specified orientations could be a particular zone axis or an extact condition of excitation (say a two beam, Bragg or 2-Bragg condition). Desktop gives the crystallographer a unique tool to help save countless hours of frustration this is the Stage Object. Access the Stage Object by the opening the Stage Object Dialog Box.

To set up Desktop Microscopist to allow tilts to be accurately determined, the following steps need to be taken:
  1. You must align Desktop Microscopist's tilt axes with the Microscope. (This operation is a one time operation for each camera length, given a stable lens system and a stable tilt holder). One method, that does work is to:
    1. put a sample into the microscope which has reasonably sharp features which move with the crystal. (The best feature would be sharp central disk HOLZ lines or a relatively sharp kikuchi band. )
    2. Set the experimental stage at (0,0).
    3. Take an exposure. (either electronically or with film.)
    4. Tilt the x-axis in a positive direction to obtain significant movement in the sharp feature but still retaining the feature of interest in the view field.(The x-axis is the axis of the holder's rod.)
    5. Take another exposure.
    6. Superimpose the images and determine angle between the direction of tilt and an horizontal line appropriate to the user.
    7. Go to the Stage Object Dialog Box. click on the Setup check box.
    8. When, the Beta box appears insert this angle into the box.
    9. Determine if the positive tilt diretion of the x-axis with respect to the y-axis gives a z-axis going into or out of the view plane(computer screen). If out of the screen, +Y must be checked.
  2. Now, you must align the sample of interest with the matrix crystal in Desktop Microscopist.
    1. Insert the crystal parameters for the experimental crystal into the Matrix Unit Cell Dialog Box .
    2. Rotate the crystal to an exact orientation, such as a zone axis using the Beam Orientation Dialog Box.
    3. Insert the experimental stage x-y tilt values into the Stage Object Dialog Box.
    4. Calculate this zone axis in Desktop Microscopist SAD/CBED object window by orienting the crystal using the Beam Orientation Dialog Box .
    5. Rotate the pattern so that the complete set of g-vectors are coincident with the calculated patterns. (You may use either the Rotate Pattern option found under the Info Menu or you can set the required rotation in the Beam Orientation Dialog Box )
    6. Congratulations, you ready to spend time in Reciprocal Space!
  3. At this point, you may want toset the Default Object Values to the current object values using the Default<->Current Dialog Box.
  4. Now, use the Stage Object Dialog Box. to calculate tilts to a pole of interest, or use the Beam Dir. Button in the Stage Object Dialog Box to calculate an exact irrational zone axis, or go an obect of interest and move the cursor around to read off addition zone axis or orientations.

ONCE THE CRYSTAL IS ALIGNED TO THE STAGE OBJECT, DO NOT USE THE BEAM ORIENTATION DIALOG BOX!!!! If, you do you will lose the alignment.
