Plot Info Dialog Box

The Plot definition dialog boxes for the SAD, Laue Back-scattered and Laue Transmission patterns are similar and can be found under the Define menu by selecting the Define Pattern... menu item or similar command.

Use the Definition Box to determine which crystal diffraction intensities will be plotted, the width of the calculated intensity spread (streaking) and whether to display the plotted intensity in the crystal's associated color or with an intensity based linear grey map.

The crystals are selected for plotting by checking the appropriate box. The Beam Orientation set in the Beam Orientation Dialog box is for the first selected crystal. If the Ppt. 1 crystal is checked but not the Matrix crystal, Desktop Microscopist will calculate the appropriate diffraction pattern with the PPt. 1 oriented parallel to the zone axis. However, if the Ppt. 2 crystal is also checked, the orientation of both crystals will be with respect to the Matrix.

Use the grey scale option to view the crystal with the spot intensity based on the relative calculated intensity. This feature is especially useful for CBED patterns where the relative intensities are easier to visualize.


The final observation with respect to this dialog box concerns the methodologies used to determine which diffraction spots will be plotted. If. the Ewald Sphere construction were held to a strict geometrical interpretation, it would be impossible to observe a zone axis pattern where rows of reciprocal lattice points can be observed. The intersection of a sphere with a plane of diffraction spots in reciprocal space defines a set of points that is smaller than typically observed. This is overcome by allowing these planes of spots to have thickness. Applying this third dimension provides the condition whereby observed diffraction patterns can be formed. The 3-D effect is due to the spreading of intensity around each reciprocal lattice point. This spreading is estimated in Desktop Microscopist using two functions:

The first function takes the stated thickness of the foil and the relation of the foil normal to the zone axis, calculates the thickness and determines a spread which is inversely proportional to the value. The second method uses the stated morphology of the precipitate crystal to calculate a streaking shape and the spread function in the desired direction.

This check box activates the crystal morphology streaking calculation. It only effects the precipitate crystals.


Extra spots and concentrations of electrons, in addition to generation from interaction with the average perfect lattice can be caused by many additional sources, inelastic interactions, double diffraction, interactions with defects and effects due to crystal size and shape and deviations due to interface interactions. (See Elastic and Inelastic Scattering In Electron Diffraction and Imaging). With the advent of verison 2.0, Desktop Microscopist, it is now possible to evaluate some of the effects caused by precipite shape and size. When the Full Streak Calcs. option is activated a Ppt. Streak Setup Dialog Box will be displayed. After setting up the calculation, Desktop will calculate a fourier transform of the Crystal Shape defined in the Crystal Shape Dialog Box for any activated precipitate crystal. The transform will be rotated to the current orientation and slices centered around the distance from the Ewald sphere to reciprocal lattice points will be displayed.
