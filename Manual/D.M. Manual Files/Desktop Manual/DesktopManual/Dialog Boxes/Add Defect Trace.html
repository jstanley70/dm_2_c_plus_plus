Add a Defect Trace Dialog Box

PATH...{Stereographic Projection Object}:Function Menu:Add a Defect Trace...

Add a Defect Trace Dialog Box

This dialog box is used when the trace of a defect is to be added to the stereographic projection. The beam direction and g are entered and checked for orthogonality. The beam direction is automatically rotated to the center of the projection and g is rotated to the top. The defect trace is added to the stereographic projection graphically by moving the cursor until the desired orientation is achieved.
