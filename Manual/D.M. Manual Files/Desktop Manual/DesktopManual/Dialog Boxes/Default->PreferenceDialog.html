Default/Preference Dialog Box

PATH...Parameter Menu: Set Default <-> Local

The Default to Local Dialog Box lets the user transfer vaules between the preference file, the default value set and the current windows values.
The arrows indicate the direction of data flow. As an example to get the data being used in the current window into the Preference files, the sequence would be:
Default <- Current button (dumps current values into the Default settings), Default -> Preferences (takes these values and dumps them into the Preferences folder).


The Default Preference Dialog Box
