Kikuchi Map Definition Dialog Box

To Set the Initial Beam orientation, use the Beam Orientation Dialog Box which can be found in the Set All Parameters Dialog Box.
Kikuchi Map:Setup Dialog:Plot Grey Scales;.i.Kossel Map:Setup Dialog:Plot Grey Scales;When this box is checked, the Kikuchi lines will be plotted with a grey scale value relative to their calculated intensity. This feature works best with a black background.
