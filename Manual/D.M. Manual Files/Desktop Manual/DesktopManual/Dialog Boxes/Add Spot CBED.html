Spot Info Dialog Box

I. Desktop Microscopist provides a method for digitally altering a calculated diffraction spot. This option provides additional flexibility in the interpretation of experimental diffraction patterns. This feature is particularly useful with patterns that are strongly influenced by the effects of dynamic (double) diffraction. In such cases, it is not inconceivable that one or two primary precipitate spots could generate a fairly complex pattern. In these cases, it can be useful to calculate a diffraction pattern based on best guess without actually knowing the orientation relationship.
1. This Dialog box allows all the important values associated with a particular diffracted spot to be set. To activate this dialog, select either Add a Spot or Move from the Info Menu. Once this is done, double clicking on a diffraction spot will cause the above dialog box to appear. The HKL value may be set to any desired value. Intensity is in percentage with a maximum value of 100..i.Spot Information Dialog:Move; .i.Spot Information Dialog:Add A Spot;


1. This pop-up menu allows the crystal associated with the spot to be set. The values of x,y and z are in reciprocal angstroms with the x-y plane being the plane of the computer screen.
