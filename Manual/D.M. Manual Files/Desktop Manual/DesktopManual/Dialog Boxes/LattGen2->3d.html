2D-3D Lattice Generation Dialog Box

This Dialog allows the user to select a set of 2-dimension lattices to calculate a potential primitive lattice. Simply, use the pop-up menu to find the Text Tables of interest. Once selection is completed click on the Calculate button and Desktop will attempt to generate a primitive lattice that aligns a three dimension point lattice with the two or more 2-dimensional lattices.
