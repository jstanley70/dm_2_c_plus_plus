Elastic Constants Dialog Box

The Elastic Constant Dialog box is similar to the Crystal Shape Dialog Box in that Desktop Microscopist uses the information in a limited number of objects. Currenlty, Elastic Constants are used for the determination of displacements about a dislocation line in the One and Two Dislocation Imaging Objects. The required values are exactly the same as described in the book by Humble and Head. The Elastic Constant Dialog Box is accessed from the Parameter Menu from the hierarchical menu Elasticity Constants.

The elastic constant matrix is a 6x6 tensor which relates the strain in a material to the stress applied. It is based on linear elastic theory or Hooke's law. The 6x6 tensor can be reduced to a 21 valued tensor when it is recognized that the tensor is symmetric around its principle diagonal, making all the bottom proportion dependent and therefore directly calculable from the upper values. More of these 21 values become dependent as the symmetry of the crystal is increased. An excellent discussion of these concepts is available in a book by Nye. The elastic constants are given in MPa. The expected tensor gives the proper values for the crystal with the <001> direction pointing out of the plane of the screen. The standard tensor is given in Cartesian coordinates. These are the expected values. For a given space group, Desktop Microscopist asks for only the required independent constants. The remainder are calculated from a knowledge of the dependencies.
