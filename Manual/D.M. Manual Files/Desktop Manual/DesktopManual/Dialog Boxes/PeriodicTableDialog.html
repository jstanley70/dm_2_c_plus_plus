Periodic Table Dialog Box

The Periodic Table is accessed in a number of locations throughout Desktop Microscopist. It is used for setting search chemistry when using the Crystal Search routines found under the Search/Match Menu and for selecting the Binary Phase Diagram Object. Its primary function is in the selection of the elements. For the Search/Match Menu, selection is set by the use of two radio buttons and multiple elements can be designated for each level (May Be, May Not Be, and Must Be Present). The Phase Diagram object has automatic selection. Clicking on a highlighted element (indicating potential binary phase diagrams) brings up another set of highlighted elements which correspond to phase diagrams which can be made with the first selected element.
1. The periodic table as used in the Atomic Info dialog box. This dialog box is used to access the information stored within Desktop Microscopist about the element of interest. Double clicking on the desired element will bring up the Atomic Info dialog box which contains a display of the selected element's stored information.
