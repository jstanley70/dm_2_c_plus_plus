Beam Definition Dialog Box: Setup for X-ray Definition

The X-ray Definition Dialog box is similar to the Electron Definition Dialog box. Differences include the need to supply a characteristic wavelength and a maximum accelerating voltage. The storage of typical characteristic wavelengths in the pop-up menu instead of standard accelerating voltages makes this a simple task. For most purposes the characteristic wavelength is the necessary value (all electron objects, the Debye-Scherrer and the diffractometer). However, the Laue transmission and Laue back-scattered objects use the accelerating voltage to determine the range of Ewald spheres which can potentially excite the reciprocal lattice.
