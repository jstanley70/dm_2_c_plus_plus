Set Background Color

The Set Background Color Dialog Box is accessed from the Edit Menu from the Set Background Color Menu item. It uses the exact same color wheel as has been previously described for both the table and plotting objects. When using monochrome mode, the background should be set to offer the maximum contrast between background and crystal.
