Crystal Color Dialog Box

PATH...Parameter Menu:Define Unit Cells:Unit Cell Dialog Box::(Color)


To set the crystal and/or the Background color, access the .i.Color:Color Wheel Dialog Box;.
The button from within the Crystal Definition Dialog Box will bring up the Color Selection Dialog Box.


Color Selection Dialog Box

This dialog box is the standard Apple Macintosh dialog box for specifying colors. Setting a color is accomplished by numerical input into the Hue Saturation Brightness fields or the Red, Green, Blue fields or alternatively by the use of a mouse. Numerical data consists of values between 0 and 65535. Values of 0 in three related fields will give black as the color and maximum values for three related fields give white as the color.
1. To use the mouse to select a color, first select the brightness by using the scroll bar on the right of the dialog box. Next, click on the desired color. Clicking on the OK button will complete the color selection.
