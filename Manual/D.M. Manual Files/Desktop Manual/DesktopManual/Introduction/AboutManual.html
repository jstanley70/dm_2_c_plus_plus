About this Manual

This manual is divided into five primary sections.

The first section consists of an introduction

    With its overall description of Desktop Microscopist, its capabilities, hardware requirements, technical support and future directions.

The second primary section is the tutorial.

    The tutorial will take the user from installation, to the primary definition functions through to actual graphical output. The user will learn how to define a unit cell, orient crystals with respect to one another, calculate and manipulate Stereographic projections and plot SAD patterns. It is suggested that the user work through the tutorial at least to the plotting of a Stereographic projection.

The third section is an in-depth look at the structure and capabilities of Desktop Microscopist.

    Read this section for an overall look at Desktop Microscopist and to achieve a greater understanding of what is possible with the available tools and to understand general techniques used in Desktop Microscopist.
The fourth part is the How to Section.
In some ways, this is an extension of the tutorial section. It is designed to help the user obtain specific information from the program.

Finally, the fifth part of this manual is description

    of every Menu Bar, Menu, Menu Item, Dialog Box, Dialog Box item and Alert which is part of Desktop Microscopist.
    This fifth part is meant to be a reference and its typical use is projected to be on an as needed basis.
