Installation of Desktop Microscopist

Before launching Desktop Microscopist take the following steps:

To Start:Decompress the compressed files

  • To do this: Copy the file labeled D.M. Compressed onto the hard drive. Double click on the icon. This will cause the following actions to occur:
  • A folder called Desktop Microscopist Folder will be created.
  • Desktop Microscopist will be stored in this folder.
  • A folder called Desktop Microscopist Crystals will be created in the Desktop Microscopist Folder with a selection of crystals.
  • A set of Phase Diagram Files will be placed in the Desktop Microscopist Folder.

Once the decompression is complete, delete the Compressed File

  • After decompression, the Desktop Compressed file may be deleted from the Hard Drive. Do not erase your master copy!!Feel free to make as many copies of Desktop Microscopist for backup purposes as desired.

Turn off the computer. Attach the hardware key

  • After placing the D.M. Init file in the system folder, turn off the computer. Next, attach the hardware key. To do this, insert the hardware key into any active ADB port. The hardware key may be connected to the keyboard, between the key board and the Macintosh, between the mouse and the keyboard or any other active ADB port. The recommended placement of the hardware key is in-line with the keyboard, plugged into the back of the CPU. Once the hardware key has been inserted into an appropriate port, the computer may be re-started.DO NOT ATTEMPT TO INSTALL THE KEY WHILE THE COMPUTER IS ON!

Installation is now complete!

  • Desktop Microscopist may be launched by double clicking on:
    • The Application Icon (looks like an electron microscope).
    • A crystal file (looks like a primitive cubic crystal with one atom per cell).
    • A Diffract Object File (looks like an electron microscope on a dog-eared piece of paper).
Double Clicking on the application Icon will open the default definition portion of the program (no Object Windows) and generate an inital preference file. Double clicking on a crystal file will open the Default Matrix Crystal Definition Dialog box and fill in the parameters with previouslhy stored information. Double Clicking on a Diffract Object File will bypass the Default Definitions and immediately open the correct window type and plot the stored graphic.
