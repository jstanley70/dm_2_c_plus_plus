Icons On The Desktop
(File Types)

The following is a summary of the icons used by Desktop Microscopist for storing and accessing data on the Macintosh desktop:
  • This is the application icon. Double click on this icon to launch Desktop Microscopist.
  • This is an object file icon. It stores the information associated with a Desktop Microscopist object.
    Double clicking on this icon will bring up the object..i.Icons:Object File;
  • This is a crystal file icon. It contains all of the information for a unit cell.
    Double clicking on this icon will bring up the crystallographic information within the Crystal Definition Dialog for the matrix.
  • This icon signifies a phase diagram.
    It contains a previously defined Phase Diagram. These files need to be stored in a single folder so that Desktop can find them.
