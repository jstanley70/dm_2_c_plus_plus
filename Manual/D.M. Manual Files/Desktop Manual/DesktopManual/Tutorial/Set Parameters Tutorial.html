Setting Plot Parameters Tutorial


Desktop Microscopist allows flexibility with respect to a number of choices when it comes to the actual graphical output of the calculated image. It is possible to control the shape of the plotted point, the color of each crystal, labeling of the crystal and whether the plot will be for a monochrome or color monitor. The user is also given control over what units structure factors are displayed in, whether Debye-Waller factors are considered, and if Desktop is to use a Bloch Wave Dynamic calculation to determine intensty.
In addition, hexagonal crystals can be labeled using either a four or a three index system. (See appendix A: The Mathematics of Desktop Microscopist for an explanation of how the conversion is calculated.)
The Control of these general issues is supported in three distinct Dialog Boxes:
  • Set Plot Shapes Dialog Box
    (Allows The User To Set the Shape of the Plotted Diffraction Pattern)
  • Set Misc. Parameters
    • (Controls hkil for hexagonal, linking, Background Color, In Color and the amount of rotation the arrow keys work)
  • Structure Factor
    • Controls how the structure factor is calculated and displayed.
  • Set Default <-> Local
    • Allows the user to transfer data into the default parameters (all new windows will have these values) and the preference file (when opening the program, default prameters will be set to the current values). Also, allows the user to retrieve default and preference file values into the current window.

Plot Parameters

  • Set the geometric figure used to represent plane poles, diffraction maximum and directions for various graphical representations. The real lattice plot is not effected by these parameters.
  • Set whether Desktop Microscopist will plot in monochrome or 8-bit color. This feature does apply to all objects. Set Plot Shapes Dialog Box
  • Set whether hexagonal planes and directions are represented by a three or four index system. Set Misc. Parameters
  • Set three levels of linkage between windows. No linking, windows are unconnected and values changed within one window do not effect other windows. Static Linking changes in one window effect other linked windows but the effect is seen only when the window is brought to the front. Dynamic Linking changes in one window are immediately displayed in all linked windows. Set Misc. Parameters
  • Activate/deactivate the use of Debye Waller factors in the calculation of structure factor values.Determine whether structure factors will be displayed in eV or Å-1. Structure Factor
  • The two buttons at the bottom (Revert to Previous Defaults and Set Defaults to Current) offer a second method for manipulating the Default settings. By clicking on Revert to Previous Defaults button the current crystals, orientation relationships etc. will be set to the stored default settings. The second button, Set Defaults to Current, will set the default settings to the current crystals, orientation relationships, etc. Set Default <-> Local
  • For certain windows, it is possible to tilt the crystal by steps using the arrow keys. The Tilt Step sets the size of these tilts in degree. Therefore, setting the Tilt Step to 5° will rotate the crystal by five degrees in the selected direction. Currently, this feature is active in the Real Lattice section. Set Misc. Parameters

Notes:

  • The two buttons in the Set Default <-> to Local Dialog Box allow the changing of all settings with a single action.Set Default <-> Local
  • CAUTION! These two buttons are irreversible! Once the defaults have been set to the current crystals, orientation relationships etc., the original set of default settings cannot be retrieved.
For purposes of this tutorial: Set the matrix plane to plot as a 'X'. Click on the button, Spot Shape - Planes and the button will change to Spot Shape - Direction. Set the matrix direction to plot as a '+'. These symbols may be changed to any particular preference at any time. Set Plot Shapes Dialog Box


After working with these four dialogs and completing the previous sections of the tutorial, the user has set all the necessary parameters to obtain useful diffraction patterns and stereographic projections. The Stereographic Projection Object will now be used to determine a stereographic projection of the matrix.
