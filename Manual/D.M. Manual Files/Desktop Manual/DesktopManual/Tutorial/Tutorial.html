3. Tutorial

This tutorial has been written to take the user through the basic steps and concepts necessary to receive maximum benefit from Desktop Microscopist. It is not intended to be a complete description of all the functionality built into Desktop Microscopist. It requires the use of other portions of the manual which contain more complete descriptions of the pertinent portion of code. After completing the Tutorial the user should be able to define crystals, set relative orientations, define electron beams, Stereographic projections and SAD diffraction patterns. In addition, the concept of Default parameters and how to control the plotting environment will be introduced.
