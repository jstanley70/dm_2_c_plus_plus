Arrow Keys


In all sections where the user is working with a set of single crystals, such as the Stereographic Projection, the SAD, HOLZ, CBED, Kikuchi Maps, Laue Backscattering, Real Lattice etc., you can use the Arrow keys to rotate the crystal by a set amount. The up-down arrow keys always move the crystals around the horizontal axis (spots move up or down) and the left-right arrow keys always more the crystal about the vertical axis (spots move left or right).

Holding down the option key when selecting an arrow key, decreases the size of the rotation by a factor of 5. Holding down the shift key when selecting an arrow key increases the size of the rotation by 5.

To set the size of the rotation, go to the Set Misc. Parameters Dialog box and fill in the Tilt Steps value with the new value) The default value is 1°.

Overbars or Negative Miller Indices:

Desktop Microscopist supports negative Miller and .i.Miller-Bravais indices; by holding the <option> key down when entering the desired number. For example, a Miller index of <1 -1 1> would be input by pressing: <1><option 1><1>. Desktop Microscopist also accepts double digit Miller indices. A normal, low index plane or direction can be entered in a standard format without delimiting values between each index. However, when any of the three or four indexes are > 9 each index must be separated by either a <,> or a <space>. Thus, entry of <1><2><option 1><1><1>(12 -1 11) cannot be interpreted by Desktop Microscopist but <1><2><space><option 1><space><1><1> will be properly interpreted as (12 -1 11).

Note: In the Stage Control Object, the above rule does not apply. Since, it was felt necessary to allow orientation to be determined by the tilt of the goniometer, this requires that fractional zone axis indecies be calculated. For this object, the user uses the standard method for inputtinga negative indecie. Thus, the <1-11> pole is entered by entering 1 <tab> -1<tab>1.

Additional Key Commands:

Diffraction Objects:Labelling Multiple Spots:

When labelling spots in a diffraction object, holding down <option>-t when clicking on a spot will label that crystal spot's to the third order, i.e. to <333>, if that spot is already labelled, the action will eliminate all spot labels to order <333>. Holding down the <option>-<shift>-t will labelled all of that crystal's spots, or delete all spot labels if the spot is already labelled.

Holding <option>-a will label all the plotted crystals spots to order <333>. Holding <option>-<shift>-a will label all the plotted crystals spots.


SAD and CBED Objects:

Generating a 2-D Lattice:

When generating a 2-D lattice found in the SAD and CBED objects, There are a number of helpful keyboard commands available to the user:

After drawing the 2 lines for generating the lattice.
Holding the <shift> key down while clicking on a line brings up a dialog box which allows the user to specify the number of g-vectors contained in the line.
Holding the <option> key down and hitting 'g' will cause a lattice to be generated.
Holding the <option> key down and hitting 'd' will delte a lattice.
Holding the <option> key down and hitting 't' will tabulate the generated lattice.

Dislocation Objects:

Holding the shift-key down while clicking on a dislocation image will bring up the defect parameters used for the calculation of that image.
