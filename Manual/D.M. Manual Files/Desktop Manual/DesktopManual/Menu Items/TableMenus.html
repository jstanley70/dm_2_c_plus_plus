Desktop Manual:Menus

Table Menus:Tables:Menus;

Tables have one menu in addition to the base set mentioned above. This menu is used to access the Set-Up Dialog for each table window. Additionally, for the d-space table, this menu allows the user to select the sort mechanism used by Desktop Microscopist to determine the order in the file, and to determine how the values are displayed.


Tables:Define Menu:Define Parameters:

Brings up the Set-Up Dialog for the respective table.

Tables:Define Menu:Sort by d-space:

Selection of this item automatically sorts the list by d-space. (This menu item works for the d-space table only.)

Tables:Define Menu:Sort by Struct. Factor:

Selection of this item automatically sorts the list by Structure Factors. (This menu item works for the d-space table only.)

Tables:Define Menu:Sort by Extinct. Dist.:

Selection of this item automatically sorts the list by Extinction Distances (This menu item works for the d-space table only.)

Tables:Define Menu:Display as d-space:

This menu item displays plane d-spacings in Å.

Tables:Define Menu:Display as Rcp. d-space:

This menu item displays the reciprocal d-space value, i.e. 1/d-space. The value is given in Å-1.

Tables:Define Menu:Display as g-vector:

This menu item displays the d-spacings as g-vector values for experimental micrographs. The value is given in millimeters and corresponds to the reciprocal d-space value multiplied by the camera constant.

Tables:Define Menu:Display as 2 Theta:

This menu item displays the Bragg diffraction angle for the given particle wavelength for the specified planes.

For all of these items, the results are displayed immediately upon selection.
