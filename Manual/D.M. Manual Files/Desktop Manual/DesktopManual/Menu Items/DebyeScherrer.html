Debye-Scherrer and Ring Pattern Menus:

Debye-Scherrer:Define Menu

Debye-Scherrer:Define Menu:Define;...:

This menu item brings up the Ring pattern definition dialog box. Use it to determine the crystals which will be diffracting, to set the criteria for the display of rings and to specify display options.

Debye-Scherrer:Define Menu:Recalculate:

Will recalculate the original diffraction pattern. Use this option to return to the original pattern.

Debye-Scherrer:Define Menu:Redraw:

Redraws the diffraction pattern. In this case, it is the same as the Refresh... under the Edit menu.

Debye-Scherrer:Define Menu:Increase Intensity;:

Increases the log dynamic range value by .5.

Debye-Scherrer:Define Menu:Decrease Intensity:

Decreases the log dynamic range value by .5.

Debye-Scherrer:Info Menu

The Info Menu is not the same for the Ring Pattern and the other SAD derived objects because the implementation of the functions must be changed in order to handle the difference in representation.

Debye-Scherrer:Info Menu:Labels:

After selecting this item, click on the ring to be labeled. A box will appear centered around the cursor. Moving the cursor, a line will appear attached to the box and the selected ring. Move the box to the desired location of the label and release the mouse.

Debye-Scherrer:Info Menu:Info:

Clicking on the ring of interest will cause information about that ring to appear in a pop-up menu at the upper left of the window. Holding the <shift> key down will display information on additional rings residing at the same location.

Debye-Scherrer:Info Menu:Add Ring:

Select this option and click on the screen. A ring will appear. Moving the mouse while holding down the button will cause the ring to move to the the desired position. Double clicking on the ring will bring up an information box. Use the spot information box to define numerical parameters. For this information box, the value stored in the x variable is the actual radius of the ring.

Debye-Scherrer:Info Menu:Delete Ring:

Click on a ring and an inverted image of the ring will be drawn to indicate that the ring has been removed. Subsequent refreshing of the screen will eliminate the selected ring.

Debye-Scherrer:Info Menu:Re-configure Ring;:

Click on a ring to select it. After selection, use the mouse to reposition the ring. Double clicking on the ring will bring up an information box. Use the spot information box to define numerical parameters. For this information box, the value stored in the x variable is the actual radius of the ring.

Debye-Scherrer:Info Menu:Move Pattern;:

Move the pattern by selecting the center portion of the pattern. Moving the mouse will alter the pattern's position within the window.

Debye-Scherrer:Exp. Info Menu

This menu activates exactly the same functions as the SAD Exp. Info. Menu.

Debye-Scherrer:Search / Match Menu;

This menu activates exactly the same functions as the SAD Search/Match Menu.
