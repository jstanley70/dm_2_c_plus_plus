Desktop Manual

Parameter Menu:

Parameter Menu:Set All Parameters:

This brings up a dialog box which allows you to define all the major parameters needed for the given Desktop Object.

Parameter Menu:Define Unit Cells:

Leads to a hierarchical menu of the four crystals. Selection of one of the crystals brings up the corresponding Crystal Definition Dialog Box. The Crystal Definition dialog boxes available via this menu may also be accessed by double clicking on the Ruler button of the desired crystal or by the command keys: <cmd-0>, <cmd-1>, <cmd-2>, <cmd-3> for the Matrix and three precipitates respectively. You can now add Debye-Waller factors to the various atomic positions.

Parameter Menu:Define Orientations:

Leads to a hierarchical menu of the three precipitate crystals. Selection of one of the crystals brings up the Orientation relationship Dialog Box. Use this to define the orientation of a set of precipitate crystals with respect to the matrix.

Define Crystal Shape:

Brings up a hierarchical menu of the four crystals. Selection of one of the crystals brings up the Crystal Shape Definition Dialog Boxx.

Parameter Menu:Elasticity Crystals;:

Brings up a hierarchical menu of the four crystals. Selection of one of the crystals brings up the Elasticity Crystal Definition Dialog Box. Use this to define the elastic constants for the respective crystal. These values are critical for the accurate calculation of Dislocation Images.

Parameter Menu:Define Beam:

Brings up Beam Definition Dialog Box that allows the user to select the type of beam and to set critical parameters such as the type of beam (x-ray,nuetron,electron), beam energy/wavelength., beam convergence, the size of the plotted diffraction pattern etc.

Parameter Menu:Define Orientation:

Brings up the Beam Orientation Dialog Box which allows reorientation of the beam with respect to the current base crystal.

Parameter Menu:Define Sample Set-up

Brings up the sample dialog box which sets the beam normal and the sample thickness. It also allows the user to set up a series of thicknesses for objects which support through thickness calculations (Dislocation Objects).

Parameter Menu: Lens Transfer Function

Brings up the Contrast Transfer Function Dialog Box which allows the definition of the contrast transfer function, calculates import defocus settings, and plots the CTF in reciprocal space for a given rotation and defocus.

Parameter Menu: Stage Control

Brings up the Stage Control Dialog Box, this is a mechnanism for setting orientation based on goniometric tilts. Important for experimental determination of primitive volumes and deterimination of tilts to new zone axis.

Parameter Menu: Structure Factor:

With version 2.0, Desktop supports various levels of sophistication with respect to the calculation of the Structure Factor for a given g-vector. Thus, it is now possible to consider the effects of temperature using Debye-Waller factors, the effects of absorption, whether the Debeye-Waller factors are to be calculated based on temperature or input by hand, whether to consider dynamic effects (not available in all objects) and whether to report the structure factor in reciprocal angstroms, voltage or as an interaction potentials. Selecting this menu item brings up the Structure Factor Dialog Box that allows the user to select the various options of interest.

Parameter Menu: Set Plot Shapes:

This brings up the Plot Shape Dialog Box which allows you to set the shape of spots seen in Stereographic Projection/SAD/Kikuchi Map etc. This allows you to differentiate spots from different crystals.

Parameter Menu: Set Misc. Parameters:

This menu item brings up the Misc. Parameters Dialog Box. With it you can control how the hexagonal crystal g-vector and direction pole indecies are displayed, either (hkl) or (hkil), whether you want linking between windows. When Linking is activated, linked windows will maintain the same orientation, crystal structures, orientation relationships etc.. Also, you can set the background color, whether the plotting is in black and white or color, and the rotation angle when using the arrow keys to rotate the crystal.

Parameter Menu: Set Default<->Local:

This menu item brings up the Set Default <-> Local Dialog Box. Use this dialog box to change the default values of Desktop Microscopist, reset the current window values to the default values or either set the stored in preference values or reset the current Default values to the stored preference values.
